# EduTrack 

O **EduTrack** é uma landing page desenvolvida para apresentar uma plataforma de gestão de aprendizado voltada para **escolas técnicas**.
O projeto foi criado com foco em uma interface moderna, clara e responsiva, destacando os principais benefícios da solução, suas funcionalidades e seu posicionamento no contexto educacional.
  
## Sobre o projeto

A proposta da EduTrack é centralizar o acompanhamento acadêmico de alunos em instituições técnicas, reunindo informações importantes em uma plataforma intuitiva e eficiente.

Esta aplicação foi desenvolvida em **React**, com foco em componentização, organização visual e construção de uma experiência agradável para apresentação do produto.

## Funcionalidades da landing page

- Apresentação institucional da plataforma
- Seção de destaque com proposta de valor
- Exibição de métricas e indicadores
- Área com pilares da solução
- Sessão de funcionalidades e diferenciais
- Apresentação da equipe
- Formulário de contato
- Layout responsivo e visual moderno

## Diferenciais apresentados no projeto

- Relatórios detalhados
- Metas de aprendizado
- Alertas inteligentes
- Gestão de parcerias
- Aplicativo do aluno
- Conformidade com LGPD

## Tecnologias utilizadas

- **React** — construção da interface e componentização  
- **TypeScript** — tipagem estática para maior segurança e escalabilidade  
- **JavaScript** — lógica da aplicação  
- **Vite** — ambiente de desenvolvimento rápido e build otimizado  
- **HTML5** — estrutura da aplicação  
- **CSS3** — estilização e responsividade  

## Objetivo do projeto

O objetivo do projeto é desenvolver uma interface web para divulgação da plataforma **EduTrack**, aplicando conceitos de desenvolvimento front-end com React, boas práticas de estruturação de componentes e construção de páginas modernas com foco em usabilidade e apresentação profissional.

## Layout da aplicação

A landing page foi estruturada para comunicar a proposta da plataforma de forma clara e estratégica, contendo:

## Como executar o projeto

### Pré-requisitos

Antes de começar, você precisa ter instalado em sua máquina:

- [Node.js](https://nodejs.org/)
- npm ou yarn

### Rodando o projeto

```bash
# Clone o repositório
git clone https://github.com/Bianca-Paiva/EduTrack-React.git

# Acesse a pasta do projeto
cd EduTrack-React

# Instale as dependências
npm install

# Execute a aplicação
npm run dev
```

Depois disso, o projeto estará disponível no navegador no endereço informado pelo terminal.
